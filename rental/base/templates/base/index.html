FAQ

Pertanyaan yang sering ditanyakan oleh pelanggan kami

Bagaimana sistem pembayarannya?

Sistem pembayaran dapat dilakukan pada WhatsApp. Jadi, setelah user mengisi formulir pendaftaran, user akan diarahkan ke WhatsApp dan memilih metode pembayaran yang tersedia. Lalu, setelah user melakukan pembayaran, user perlu melakukan konfirmasi kepada admin.

Apa saja yang diperlukan untuk merental mobil?

Untuk dapat merental mobil, user perlu mengisi formulir pendaftaran yang tersedia pada website ini.

Berapa batas usia untuk merental mobil?

Batas minimal usia untuk merental mobil adalah 17 tahun atau minimal sudah memiliki KTP karena pada saat merental mobil akan dimintai KTP sebagai identitas perental.

Apakah saya bisa merental mobil untuk teman atau orang lain?

Pada website kami, user bebas menggunakan mobil untuk hal apapun asalkan dengan mengikuti syarat dan ketentuan yang berlaku.
